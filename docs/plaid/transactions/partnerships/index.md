# Transactions partners

#### Find partners who work with Plaid Transactions

=\*=\*=\*=

#### Overview

The following is a list of companies who have partnered with Plaid to provide Transactions capabilities via a special processor token, along with instructions on how to enable those integrations.

| Partner | Description |
| --- | --- |
| [Boom](/docs/transactions/partnerships/boom/) | Report your customers' rent payments to Experian, Equifax, and TransUnion using the bank account they've linked via Plaid in your platform |
| [Cardlytics](/docs/transactions/partnerships/cardlytics/) | Turn your customers' everyday purchases into personalized cashback rewards with the Cardlytics Rewards Platform |
| [Knot](/docs/transactions/partnerships/knot/) | Instantly make your card top-of-wallet at merchants with Knot's API |
| [Layer](/docs/transactions/partnerships/layer/) | Offer SMB accounting embedded directly within your platform. |
| [Sardine](/docs/auth/partnerships/sardine/) | Protect your users by using Sardine's all-in-one fraud and compliance API |
| [Teal](/docs/transactions/partnerships/teal/) | Teal allows SMB vertical SaaS and neobanks to build and offer their own branded accounting suite to their SMB customers |
| [Esusu](/docs/transactions/partnerships/esusu/) | Report positive-only rent payments to Experian, Equifax, and TransUnion with Esusu and Plaid |
| [Pinwheel](/docs/transactions/partnerships/pinwheel/) | Activate new accounts with the industry’s top performing direct deposit and bill switching solution, The Pinwheel Switch Kit. |

For more information, see the full list of partners in the [Plaid Dashboard](https://dashboard.plaid.com/developers/integrations).
